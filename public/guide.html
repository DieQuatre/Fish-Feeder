Setup Guide

Set up your automatic fish feeder system step by step

🔧 Parts ⚡ Wiring 👤 Account 💻 Arduino 🎯 Usage
1

🔧 Required Parts

You need the following components to build the system:

📡
ESP8266 (NodeMCU)
WiFi-enabled microcontroller
⚙️
SG90 Servo Motor
Opens and closes the feed gate
📏
HC-SR04 Distance Sensor
Measures food level
🔌
Jumper Wires
Connection cables
ℹ️ It is recommended to use an external 5V power supply for the servo motor.
2

⚡ Wiring Connections

Connect the components according to the table below:

PartPinESP8266 Pin
HC-SR04 TrigTrigD1
HC-SR04 EchoEchoD2
HC-SR04 VCCVCC5V
HC-SR04 GNDGNDGND
Servo MotorSignalD3
Servo MotorVCC5V
Servo MotorGNDGND
⚠️ Unplug the ESP8266 before making connections.
3

👤 Create Account & Add Device

  1. Go to the home page and click Sign Up
  2. Set a username (min 3 chars) and password (min 6 chars)
  3. After signing in, click the ➕ Add Device button
  4. Give your device a name (e.g. "Living Room Aquarium")
  5. The system will give you a Device Token — copy it!
ℹ️ Click the eye icon 👁️ to reveal the token, click 📋 to copy it.
4

💻 Upload Arduino Code

Open Arduino IDE and follow these steps:

  1. Download Arduino IDE: arduino.cc
  2. Add ESP8266 support from Board Manager:
// File → Preferences → Additional Board Manager URLs:
http://arduino.esp8266.com/stable/package_esp8266com_index.json
  1. Install this library from Library Manager:
ArduinoJson by Benoit Blanchon
  1. Open the firmware/fish_feeder.ino file
  2. Fill in these 3 lines with your own info:
// Your WiFi credentials
const char* WIFI_SSID     = "YOUR_WIFI_NAME";
const char* WIFI_PASSWORD = "YOUR_WIFI_PASSWORD";

// Token copied from dashboard
const char* DEVICE_TOKEN  = "DEV-XXXXXXXXXXXXXXXX";
  1. Update the SERVER_URL with your Render address
  2. Select Board: NodeMCU 1.0 (ESP-12E Module)
  3. Select the port and click Upload
⚠️ Make sure the ESP8266 is connected via USB during upload.
5

🎯 Dashboard Usage

Everything is ready! Here's what you can do on the dashboard:

  • 🟢 Device Status: See if your ESP is online or offline
  • 📊 Food Level: Track how much food is left as a percentage
  • 🐟 Feed Now: Feed remotely with the press of a button
  • 📜 Feed History: See when and how feeding occurred
  • 🕐 Last Feed: Check the latest feeding time
ℹ️ The dashboard updates every 5 seconds.

❓ FAQ

  • How does the food level sensor work?
    The HC-SR04 distance sensor is mounted on top of the food container.
  • I want to change WiFi, what should I do?
    Update the WIFI_SSID and WIFI_PASSWORD values in the Arduino code.
  • The device shows "Offline", what should I do?
    Check the WiFi connection, power supply, and SERVER_URL.
  • Can I add a second device?
    Yes! Create a new token from the dashboard.
